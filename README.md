# Bundesliga Coach Info Bot

## Overview
This project is a command-line Python script that retrieves the current coaches of German Bundesliga football clubs by city or club name. It uses Wikidata and Wikipedia APIs for up-to-date information and processes natural user queries dynamically.

## Requirements
- Python 3.8 or higher
- requests library (`pip install requests`)
